## 📦 **Installation & Deployment**

### **⚡ Method 1: Auto-Deploy (Recommended - 2 Minutes!)**

**Easiest way - Zero configuration needed!**

1. **Fork this repository** (Click "Fork" button on GitHub)
2. **Enable GitHub Pages**:
   - Go to Settings → Pages
   - Source: Select "GitHub Actions"
   - Save
3. **Wait 1-2 minutes** for automatic deployment
4. **Done!** Your app is live at: `https://YOUR-USERNAME.github.io/joy-ai-master/`

**Every time you commit**, it automatically redeploys! 🎉

### **Method 2: Direct Download**

1. Download `index.html`
2. Open in any modern browser
3. Start chatting!

### **Method 3: Local Development**

```bash
# Clone the repository
git clone https://github.com/YOUR-USERNAME/joy-ai-master.git

# Navigate to folder
cd joy-ai-master

# Open with browser
open index.html
# or run a local server
python -m http.server 8000
# Then visit: http://localhost:8000
```

## 🎯 **How to Use**

### **Basic Usage:**

1. **Open the app** in your browser
2. **Click "New Chat"** to start a conversation
3. **Type your question** or coding request
4. **Select a model** from the right panel (Opus/Sonnet/Haiku)
5. **Get instant responses** with syntax-highlighted code

### **Advanced Features:**

**🎤 Voice Input:**
1. Click the **🎤** microphone icon
2. Speak your question (supports English & Bengali)
3. Your speech is automatically transcribed
4. Press send or edit before sending

**▶️ Run Code:**
1. Ask Claude to write code (JavaScript, Python, or HTML)
2. Click the **▶️ Run** button on the code block
3. See output instantly below the code
4. JavaScript runs in browser
5. Python runs via Pyodide (takes a moment to load first time)
6. HTML opens in preview window

**📎 Upload Files:**
1. Click **📎 Attach File** button or drag & drop files
2. Select one or multiple files
3. Text files are automatically read
4. Reference files in your conversation
5. Claude can analyze and work with the file content

**💬 Streaming Responses:**
1. Toggle in Settings or top toolbar
2. See Claude's response word-by-word in real-time
3. More engaging chat experience

**🌙 Theme Toggle:**
1. Click **🌙/☀️** icon in top bar
2. Switch between Dark and Light themes
3. Preference is saved automatically

**🌐 Language Switch:**
1. Select language from dropdown (English/বাংলা)
2. UI updates instantly
3. Voice input adapts to language

**📥 Export Chat:**
1. Click **📥** icon in top bar
2. Download conversation as JSON
3. Share or backup your chats

**⚙️ Admin Panel:**
1. Click **⚙️** icon in top-right
2. Enter password: `admin123` (change this!)
3. View usage statistics:
   - Total messages sent
   - API calls made
   - Code executions run
   - Active chats count
4. Change admin password
5. Configure settings

## ⚙️ **Configuration**

### **Change Admin Password:**

Edit line in the HTML file:
```javascript
let adminPassword = 'admin123'; // Change this to your secure password
```

### **Set Default Model:**

```javascript
let selectedModel = 'claude-opus-4-6'; // or 'claude-sonnet-4-6'
```

### **Customize Theme:**

Edit CSS variables in the `:root` section:
```css
:root {
    --bg-dark: #0a0a0a;
    --accent: #00d9ff;
    /* Customize colors here */
}
```
